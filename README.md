# 🫀 Heart Disease Prediction System

An AI-powered medical decision support system that predicts the risk of heart disease using clinical data and machine learning.

## 📊 Project Overview

This project provides an end-to-end machine learning solution for heart disease risk prediction.

It combines:
- Data analysis (EDA)
- Feature engineering
- Model training
- Real-time prediction
- Interactive web application

The system helps simulate early risk screening based on patient clinical indicators.

---

## 🎯 Business / Medical Problem

Early detection of heart disease is critical but often challenging.

Healthcare professionals and patients need:
- Quick risk assessment tools  
- Data-driven insights  
- Support systems for early screening  

This project provides a predictive system that estimates heart disease risk using patient data.

## 🧠 Key Features

### 🫀 AI Prediction System
- Predict heart disease risk  
- Probability-based output  
- Real-time input form  

---

### 📊 Exploratory Data Analysis (EDA)
- Target distribution analysis  
- Age distribution  
- Cholesterol vs disease  
- Heart rate vs disease  
- ST depression analysis  
- Correlation heatmap  

---

### 🧪 Feature Engineering
- Heart stress feature  
- Age grouping  
- Derived clinical indicators  

---

### 📈 Model
- Logistic Regression  
- Feature selection applied  
- ~75% accuracy

  - Prediction
